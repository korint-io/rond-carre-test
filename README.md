# Test de personnalité rond-carré

Un test de personnalité interactif qui détermine si vous êtes plutôt "rond" (décontracté, spontané) ou "carré" (organisé, méthodique).

## Description

Ce test comporte 22 questions sur vos habitudes quotidiennes, votre organisation et votre comportement. À la fin, vous recevrez un score qui révèlera votre profil de personnalité.

## Fonctionnalités

- 22 questions avec des réponses à choix unique ou multiple
- Système de scoring avec valeurs positives et négatives
- Interface responsive adaptée aux mobiles et tablettes
- Design moderne avec animations et transitions
- Barre de progression
- Résultats détaillés avec interprétation

## Technologies

- HTML5
- CSS3 (avec animations et responsive design)
- JavaScript vanilla (sans framework)

## Utilisation

Ouvrez simplement le fichier `index.html` dans votre navigateur ou visitez la version en ligne sur GitHub Pages.

## Déploiement

Ce projet est automatiquement déployé sur GitHub Pages via GitHub Actions.
